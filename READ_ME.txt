fil-forklaringer:

- test.py er pythonscriptet som sender verdiene til porter.json. Den bruker en api for å finne verdiene.
- porter.json er en json som får verdier fra test.py, og visuelt%20copy.html tar verdiene og bruker dem.
- visuelt%20copy.html er selve nettsiden. Den bruker javascript for å hente verdier fra porter.json, og så bruker dem til å sette nåler på ett kart og en popupp som dukker opp når man trykker på nålene. Bruker openstreetmap api.
- chatterngptern.py er ett script som var laget av chat gpt. Jeg brukte den som referanse på noen ting.
- backup.html er akkuratt det som det høres ut som. Det er en backup av en tidligere versjon av nettsiden
- whaterverthefuck.csv er en csv som jeg brukte som referanse for oppbyggningen av json filen.
- ikke%20under%20bruk mappen er mappen hvor jeg har noen av filene til api oppgaven vi hadde i fjor.
- terminal.py er en uferdig versjon av det videre steget for å gjøre programmet full automatisert.
- ugangster.bat. jeg brukte denne filen som ett eksperiment før at jeg begynnte å bruke os i python.
